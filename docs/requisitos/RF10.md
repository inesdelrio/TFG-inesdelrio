# RF10 - Busqueda y filtros de eventos

## Estado del requisito

Cerrado y validado en entorno local.

## Objetivo

Permitir que el listado publico de eventos se ajuste a criterios basicos de busqueda para que el usuario pueda localizar actividades por fecha, entidad organizadora o ubicacion.

## Alcance funcional

- El listado `GET /eventos` admite filtros por:
  - `eventDate`
  - `entity`
  - `city`
- Los filtros se leen desde la query string.
- Los filtros se mantienen al recargar la pagina.
- Existe accion para limpiar filtros y volver al listado completo.
- El sistema muestra un estado especifico cuando no hay resultados.

1. Ejecutar `npm run dev`.

3. Aplicar uno o varios filtros desde el formulario.

5. Navegar por la paginacion si existen varias paginas.
6. Pulsar `Limpiar filtros` y comprobar la vuelta al listado completo.
7. Probar un filtro sin coincidencias y verificar el mensaje de no resultados.
